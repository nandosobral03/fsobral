# Build/Test Commands

- dev: `npm run dev`
- build: `npm run build`
- lint: `npm run lint`

# Code Style

- **Framework**: Next.js (App Router), React, TypeScript
- **Styling**: Tailwind CSS
- **Naming**: PascalCase for components, kebab-case for files/directories
- **Imports**: `@/` alias for src root
- **Fonts**: Roboto, Newsreader, Outfit, Roboto Condensed (configured in layout)
- **Components**: Functional components, typed props
- **Formatting**: Prettier (inferred from standardized indentation)
